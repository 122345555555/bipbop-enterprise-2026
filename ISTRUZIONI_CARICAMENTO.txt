HOTFIX CACHE + REPORT DI TUTTE LE OFFERTE

Sostituisci questi 2 file nel repository GitHub:

1. index.html
   Va nella cartella principale del repo.

2. src/parser.js
   Va nella cartella src.

Perche serve anche index.html:
- il browser/Vercel potrebbe usare ancora src/parser.js?v=1.0
- questo file cambia gli script a ?v=1.1
- cosi l'app e costretta a scaricare il parser aggiornato

Cosa risolve:
- riconosce "Report di tutte le offerte"
- lo mette in Inventario
- riconosce anche "All Listings"

Dopo il commit GitHub:
1. aspetta il deploy Vercel
2. apri l'app
3. fai Cmd + Shift + R
4. vai su Import Amazon
5. ricarica il report

Destinazione corretta:
Inventario

Se ancora non va:
- in Import Amazon scegli manualmente "Inventario" nel menu Destinazione import
- poi ricarica il file
