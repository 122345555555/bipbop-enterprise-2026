ISTRUZIONI

1. Carica tutti i file su GitHub nel repository bipbop-enterprise-2026.
2. Esegui in Supabase:
   sql/schema_bipbop_enterprise_2026_v1_0.sql
3. Attendi il deploy Vercel.
4. Apri il sito e fai Cmd+Shift+R.
5. Vai in Setup e inserisci Supabase URL e Publishable Key.
6. Vai in Diagnostica: deve comparire SOLO TABELLE BB100 GROWTH ENGINE.
7. Importa i report Amazon.

Nota:
questa versione usa un nuovo spazio dati bb100_.
I dati delle vecchie release restano in Supabase ma non vengono letti da questa versione.
